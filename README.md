# Usages

## Gradle

### Add the JitPack repository to your build file
```
allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
	}
}
```

### Add the dependency
```
dependencies {
    implementation 'com.github.xici:AndroidSimpleTipDialog:1.0.0'
}
```
